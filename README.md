# rollup + workspace 구성안

- init

  ```sh
  $ yarn run bootstrap
  # $ yarn run bootstrap:install # node_modules 를 설치
  # $ yarn run bootstrap:builds # workspace 의 패키지 빌드
  ```

- package build

  ```sh
  $ yarn run bootstrap:builds
  ```

- demo run

  ```sh
  $ yarn workspace @jood-starter-rollup/demo serve
  ```
